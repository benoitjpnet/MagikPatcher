Pour patcher sous linux, dézipper le script de patch qui vous convient
(bonne version v1 ou V2, ...) dans le dossier qui contient les vidéos.
Ouvrez une console, aller dans le dossier des vidéos.
Lancez le script ./nomduscript

Voilà
